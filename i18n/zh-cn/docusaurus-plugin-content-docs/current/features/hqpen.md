## 如何使用

1. 点击菜单栏中的“其他”。
2. 点击“设置”
3. 找到 '播放器' -> '高清画笔', 设置它。
4. 生效。

## 注意
1. 我们计划于在 ClipCC 3.2 正式发布此功能。
2. 该功能可能会影响项目的渲染效率。
